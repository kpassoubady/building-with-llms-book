# Appendix C: Setup & Installation

## Prerequisites

- Python 3.11+ (3.12 recommended)
- VS Code with an AI coding assistant extension
- At least one LLM API key (OpenAI, Google Gemini, or Anthropic Claude)
- Git installed

## Required Packages

```
litellm
openai
tiktoken
python-dotenv
google-generativeai
anthropic
faiss-cpu
chromadb
flask
fastapi
numpy
requests
jupyter
```

## Automated Setup & Verification

For detailed, platform-specific installation scripts (macOS and Windows) and automated verification steps, please use our dedicated setup repository:

**[gen-ai-setup](https://github.com/kpassoubady/gen-ai-setup)**

This repository provides scripts that will automatically configure your Python environment, install the required packages, and verify that your LLM API keys are working correctly.
